## Shell Scripting Concepts

This repository contains basic to intermediate Shell Scripting concepts with practical script examples.
Each script is numbered for easy understanding and follows a structured learning path.

---

📘 Topics Covered

- Variables
- User Input
- Conditional Statements ("if-else")
- Loops ("for", "while")
- Arrays
- Functions8
- File Handling
- String Operations
- ...
---

🚀 How to Run Scripts

1. Give permission:

chmod +x script_name.sh

2. Run the script:

./script_name.sh
