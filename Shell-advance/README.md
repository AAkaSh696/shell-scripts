## Shell Scripting - Advanced

This folder contains advanced shell scripting tasks completed as part of the Days 16–21 of the 90 Days of DevOps Challenge.

These scripts focus on real-world automation use cases and help in building practical DevOps skills.

---

📘 Topics Covered

- Backup Automation
- Disk Usage Monitoring
- Conditional Logic in Real Scenarios
- Package Installation Automation
- Input Validation
- System Health Checks

---

📂 File Structure

Each script represents a real-world task:

backup.sh              # Automates backup of files/directories
check_number.sh        # Checks number type (even/odd, positive/negative)
disk_check.sh          # Monitors disk usage and alerts
install_package.sh     # Installs required packages automatically
process_monitor.sh     # Checks running processes
log_monitor.sh         # Monitors logs and extracts useful info

---

🚀 How to Run Scripts

1. Give execute permission:

chmod +x script_name.sh

2. Run the script:

./script_name.sh
